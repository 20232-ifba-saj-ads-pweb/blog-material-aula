# JavaScript

O JavaScript (JS) é uma linguagem de programação leve, interpretada ou  Compilação just-in-time (JIT) com funções de primeira classe. Embora seja mais conhecido como o idioma de script para páginas da web, muitos ambientes não navegadores também o usam, como Node.js, Apache CouchDB e Adobe Acrobat.O JavaScript é um idioma dinâmico de múltiplos paradigma, com base em protótipo, de tiro único, suporta os estilos orientados a objetos, imperativos e declarativos (por exemplo, programação funcional).
